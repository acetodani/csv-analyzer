Frequently asked questions

What file types are supported?

CSV files only (.csv). Make sure your file has a header row as the first line — column names are used as chart titles.

Is my data secure?

Yes. Everything runs entirely in your browser using JavaScript. No data is ever sent to a server. You can even use this tool offline after the page has loaded.

What happens with large files?

Files with more than 100,000 rows are automatically sampled to keep the tool fast and responsive. A banner will notify you when sampling is active.

Which columns get charts?

Numeric columns get histograms with stats (min, max, mean, median). Categorical columns with up to 30 unique values get bar charts. Date columns get a line chart grouped by year or month.

Can I share my charts?

Yes — use the "Copy link" button in the dashboard to generate a shareable URL that encodes your data. Anyone with the link can view the same charts without uploading a file.
